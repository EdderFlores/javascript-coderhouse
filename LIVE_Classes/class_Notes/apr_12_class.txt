QUE ES JS?
Es Lenguaje de programacion, usado principalmente para aportar DINAMISMO a sitios y webapps
Es un lenguaje de programacion INTERPRETADO (primer LP que se ejecuta nativamente en un navegador)

- JS Creado en 1997

Java y JS son lenguajes de programacion que comparten base de su sintaxis, el uso de cada uno es distinto.
    Java funciona a traves de delcaraciones que se establececn a partir de un sistema de clases en tiempo de compilacion
    JS se basa en un sistema de tiempo de ejecucion donde cada dado distinto puede representar valores booleanos, de cadena o numericos

Se le llama "Peticion" a la comunicacion de front a Back y se llama respuesta de back a front
"Ejemplo: Restaurante, el back-end es la cocina, el usuario se comunica a traves de meseros para hacer peticiones a la cocina."

COMO DECLARAR VARIABLES:
    Hay 3 acciones principables al manejar variables: (Es opcional usar "" o '')
        Declarar: Creacion de variable vacio
        Asignar valor: guardar un dato dentro de la variable
        Inicializar/invocar/llamar: Mencionarla, traerla con proposito de utilizarla o visualizarla

LET y CONST:
    Las declaraciones con 'let' y 'const' tienen controles adicionales para las variables. 
    Una variable 'let' puede recibir multiples asignaciones en el transcurso de la aplicacion, puede cambiar de valor varias veces.
    Una variable 'const' recibe una unica asignacion al momento de su declaracion, impidiendo que su valor se modifique luego.
